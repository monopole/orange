Vanadium

Vanadium lets distributed services find each other, securely communicate, and delegate authority using public-key cryptography.

I contributed to the Go libraries and overall project organization, wrote extensive, tested tutorials, worked closely with UX research to validate our material and built demos. One such demo is volley, a distributed OpenGL app for iOS, Android, OS X, and X11 (linux) written in Go that demonstrates vanadium’s cross platform power (code).

Vanadium and kubernetes (k8s) started about the same time, approaching distributed computing from different directions. K8s is a centrally controlled orchestration system that runs existing software, while Vanadium is a set of libraries and tools to build new microservices - on PC, mobile, and IoT, that have no option other than to securely coordinate with each other. Vanadium baked strong crypto directly into RPC and discovery, predating service mesh concepts that are add-ons to kubernetes. Google’s investment into V was halted in favor of fuchsia and k8s.
