cluster configuration management

Google 2020

Kustomize is an open-source cluster configuration management tool for kubernetes.

I founded the project, and led development from nothing to 6500 github stars and over 100 contributors.

In kubernetes, one declares the state of all cluster objects in files, then applies those files to the cluster, where controllers seek to drive objects to their newly desired states. To change the cluster one edits these files and reapplies them. This creates a new problem: how to manage these edits, how to manage hundreds if not thousands of cluster variants. Kustomize solves this by declaring the edits themselves as kubernetes objects.

secure, distributed computing

Google 2016

Vanadium lets distributed services find each other, securely communicate, and delegate authority using public-key cryptography. The project predated service mesh concepts bolted on to kubernetes years later.

I contributed to the libraries and overall project organization, wrote extensive, tested tutorials, and built demos. One such demo is volley, a distributed openGL app for iOS, Android, OS X, and X11 (linux) in Go that demonstrates vanadium’s cross platform power (code).

Investment into Vanadium was halted in favor of fuchsia. It was an honor to work with a team including Dave Presotto and Ken Thompson, building the first large system based on Pike’s Go.

video chat as a small business platform

Google 2014

Google Helpouts was Google’s attempt to turn video chat into a small business platform. If a Google query (fix a leaking toilet?) failed to offer quality results, Google could offer the user a chance to buy time with a human expert, anywhere in the world.

I worked on the Android app and the API it talked to, arranged for consumer and provider notifications, and did the Google search failed query integration.

This project offered piano lessons, cooking, vacation advice and medical services in one app. All of this in one UX was too broad and new for both the consumers and the providers. One-medical is an example of how to do this well in one vertical. It would have been wildly successful had it launched during a pandemic, but didn’t cross the chasm in 2014.
